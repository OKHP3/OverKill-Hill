OverKill Hill P³™ started with the feeling that if something is worth doing, it’s worth overdoing! To explore every branch and turn of the rabbit-hole. We share the results of that exploration, the protocol-driven promptcraft, and the systems we build along the way.

OKHP³™ began organically, first there was a desire to write a book, using AI... Then a realization it wasn't possible, or least within my capabilities at that time. So I started building tools to help me do it, and then building more tools to help build those tools.

Like Forest Gump's run across town, turned into an ultramarathon criss-crossing the country... before long, I had a whole ecosystem of custom GPTs, and now I can hardly remember... why I started "running" in the first place ...I am going to get back to writing that book ... but there's one more GPT to create, prompt to craft, or possibility to explore first...
