# Brand image library

Generated brand-image variants and unused source renders, archived from the
live tree on 2026-05-03 by `scripts/move_orphans_to_library.py`. None of
these files are referenced by any production page.

**Disposition options:**

- Keep here as a media kit (current state).
- Exclude from deploy by adding `assets/img/library/` to your build's
  ignore list (Cloudflare Pages: `.cfignore`; Netlify: handled by
  `_redirects` or build settings).
- Delete entirely: `rm -rf assets/img/library/`.
