Found‑Rᵧ

GPTs that build GPTs: the recursive heart of our ecosystem.

About Found‑Rᵧ

Found‑Rᵧ is a meta‑framework dedicated to creating tools that can assemble and evolve other GPTs. By treating each agent as a component with inputs and outputs, we can chain behaviours, coordinate tasks and generate new capabilities on demand.

Underlying GPTs

  • Protocol Compiler: Turns structured specifications into working prompt protocols.
  • Agent Builder: Wraps multiple GPTs into a single orchestrated agent with memory and context management.
  • Recursion Monitor: Watches chains of GPT calls and suggests optimisation or termination strategies.
  • Emergent Designer: Experiments with novel configurations and reports on their behaviours.
